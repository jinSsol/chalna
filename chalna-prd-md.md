# PRD: 찰나 (Chalna) - 반응속도 테스트

## 1. 개요

### 1.1 제품 비전

간단하고 재미있는 반응속도 테스트 앱. 측정 결과를 친구들과 공유하며 경쟁하는 재미 제공.

### 1.2 목표 사용자

- 게임을 좋아하는 10~30대
- 자신의 반응속도가 궁금한 사람
- 친구들과 경쟁하고 싶은 사람
- 짧은 시간에 즐길 수 있는 앱을 찾는 사람

### 1.3 핵심 가치

- **심플함**: 설명 없이 바로 플레이 가능
- **재미**: 등급 시스템과 공유로 경쟁 유도
- **중독성**: 더 좋은 기록을 위한 반복 플레이

## 2. 기능 명세

**화면 구성**

```
┌─────────────────────────────┐
│            찰나              │
│     ⚡ 반응속도 테스트         │
│                             │
│      ┌───────────────┐      │
│      │               │      │
│      │   🎯 TAP      │      │
│      │               │      │
│      └───────────────┘      │
│                             │
│   ┌ 최고 기록: 187ms (🚀) ┐  │
│   └───────────────────────┘  │
│                             │
│  [클래식] [랜덤] [연속]       │
│                             │
│  ┌─────────────────────┐    │
│  │ 🏠홈  📊기록  ⚙️설정 │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
```

> 글래스모피즘 UI: 반투명 카드 + 블러 배경 + 라벤더~바이올렛 그라데이션
> 하단 탭 바: 플로팅 스타일, 홈/기록/설정 3개 탭으로 전환

**기능 상세**

| 기능 | 설명 | 우선순위 |
| --- | --- | --- |
| 시작 버튼 | 선택된 모드로 게임 시작 | P0 |
| 모드 선택 | 클래식/랜덤/연속 탭 | P0 |
| 최고 기록 | 개인 베스트 표시 | P1 |
| 하단 탭 바 | 홈/기록/설정 전환 (글래스 스타일) | P0 |

### 2.2 게임 화면 (클래식 모드)

**흐름**

```
[대기] → [준비(빨강)] → [신호(초록)] → [결과]
         1~5초 랜덤      탭 감지
```

**화면 상태**

| 상태 | 배경색 | 텍스트 | 동작 |
| --- | --- | --- | --- |
| 대기 | 회색 | "화면을 탭하여 시작" | 탭 시 준비로 |
| 준비 | 빨강 | "기다리세요..." | 탭 시 실패(너무 빠름) |
| 신호 | 초록 | "지금!" | 탭 시 결과로 |
| 결과 | 배경 | 결과 카드 | - |

**너무 빠른 탭 처리**

```
┌─────────────────────────────┐
│                             │
│         ❌ 너무 빨라요!       │
│                             │
│     초록색이 될 때까지        │
│        기다리세요            │
│                             │
│         [다시하기]           │
│                             │
└─────────────────────────────┘
```

### 2.3 결과 화면

**화면 구성**

```
┌─────────────────────────────┐
│                             │
│         ⚡ 번개급!           │
│                             │
│      ┌───────────────┐      │
│      │               │      │
│      │    142 ms     │      │
│      │               │      │
│      └───────────────┘      │
│                             │
│    🏆 새로운 최고 기록!       │
│                             │
│   [다시하기]    [공유하기]    │
│                             │
└─────────────────────────────┘
```

**기능 상세**

| 기능 | 설명 | 우선순위 |
| --- | --- | --- |
| 반응 시간 | ms 단위 대형 표시 | P0 |
| 등급 | 시간에 따른 등급 + 이모지 | P0 |
| 최고 기록 알림 | 신기록 시 표시 | P1 |
| 다시하기 | 같은 모드로 재시작 | P0 |
| 공유하기 | 결과 이미지 SNS 공유 | P1 |

### 2.4 연속 모드

- 5회 연속 테스트 진행
- 각 라운드 결과 표시
- 최종 결과: 평균값 + 최고/최저

```

```

| 등급 | 이모지 | 반응 시간 | 설명 |
| --- | --- | --- | --- |

| 느림 | 🐢 | > 300ms | 연습 필요 |

## 4. 비기능 요구사항

### 4.1 성능

- 반응 시간 측정 정확도: ±5ms 이내
- 앱 시작 시간: 2초 이내
- 게임 시작 지연: 없음

### 4.2 안정성

- 앱 크래시율: 0.1% 미만
- 백그라운드 진입 시 게임 일시정지

### 4.3 호환성

- Android: 6.0 (API 23) 이상
- iOS: 12.0 이상
